# Editorial Writing Guidance

## 1) Voice and claims
- Trust-first, factual, concise.
- Avoid exaggerated performance claims.
- If a statement can be misunderstood as financial guarantee, add explicit qualification.

## 2) Page anatomy for SEO + GEO
- Opening summary: 2-3 sentences that answer "what this page is about".
- Heading map: one H1, then intent-focused H2 sections.
- Evidence layer: data points, dates, author/source context when relevant.
- Action layer: related internal links to service, support, legal, or contact pages.

## 3) Metadata writing standards
- Meta title: target <= 60 chars, include primary intent and entity name.
- Meta description: target <= 155 chars, explain value and context.
- Slug: short, human-readable, stable.
- OG image: must exist for all shareable pages.

## 4) GEO extraction standards
- Keep the first paragraph self-contained and factual.
- Use explicit nouns (brand, product, regulation, market, date) instead of vague references.
- For FAQ-like pages, keep answers direct and short in the first sentence.
- For research/news, include publish date and author whenever available.

## 5) Internal linking standards
- Every new article should link to at least 2 relevant internal pages.
- Prefer contextual anchor text over generic "click here".
- Link flow example: service -> research -> support -> contact.

## 6) Pre-publish checklist
- Unique title + meta description.
- Clean slug.
- Featured image has alt text.
- Category + tags are set.
- Author and publish date are set (for article/report content).
- Related links included.
- Legal/disclaimer block included if needed.
